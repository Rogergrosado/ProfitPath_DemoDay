Styling Philosophy for ProfitPath
🔳 Overall Visual Style
Mode: Default to Dark Mode with Light Mode toggle.

Feel: Professional, minimalist, high-contrast with vibrant accents.

UI Paradigm: Card-based + component modularity (inspired by tools like Notion, Linear, Stripe Dashboard).

🖼️ UI Tokens & Color Palette
Purpose	Dark Mode	Light Mode
Background	#0d0f13 (Deep Navy)	#ffffff (Clean White)
Card/Modules	#222831 (Dark Charcoal)	#f9f9f9 (Ultra-light gray)
Primary Accent	#fd7014 (Vibrant Orange)	#fd7014
Hover Accent	#fd8f47 (Light Orange)	#fd8f47
Typography (main)	#ffffff	#0d0f13
Typography (secondary)	#b0b3b8	#37414f
Borders / Dividers	#37414f	#dcdcdc

🧱 Component Styling Strategy (Per Page)
🏠 Landing Page
HeroSection: Centered typography with animated gradient text (orange highlight), clean CTA button (bg-orange-500 hover:bg-orange-400)

Features: Icon grid (monochrome + orange hover accents), Tailwind grid-cols-2 md:grid-cols-4

ProfitCalculator: Use useState, with live input updates, styled sliders and cards inside a soft-shadow container

🔐 Auth Page
Tabs: Use @headlessui/react for animated tab transitions (tab glow in orange)

Forms: Rounded, full-width inputs (bg-slate-800, focus:ring-orange-500)

Submit Button: bg-orange-500 text-white hover:bg-orange-400

📊 Dashboard Page
KPISection: Four-card grid (rounded-xl, bg-charcoal, border-slate-600)

SalesByCategoryChart: Use recharts styled with stroke: #fd7014, fillOpacity: 0.2

GoalProgressBar: Horizontal progress indicators (bg-slate-600, fill-orange-500), with subtle tooltips

📋 Watchlist Page
Table: react-table with sticky headers, dark cell borders, light-on-dark font

Modals: Slide-up bottom modals (mobile-friendly), form elements with subtle orange focus rings

Status Badges: Use bg-slate-700 + text colored conditionally (text-green-400, text-yellow-400)

📦 Inventory Page
InventoryTable: Zebra row styling (bg-slate-800 / bg-slate-900)

ReorderButton: Icon button with hover animation (transition-all, scale-105)

EditInventoryModal: Grid form layout with toggle fields

📅 Reorder Calendar
Use: react-calendar or @fullcalendar/react with custom dark theme

Modal: On event click, open InventoryDetailModal with inventory card-style layout

📈 Performance Analytics
PerformanceKPIs: Animated counters with trend arrows (red/green)

CategoryBreakdownChart: Pie charts with orange/white/gray slices

ManualSalesModal: Floating modal with dimmed background and form controls

📊 Reports Page
ReportEditor: Drag-and-drop zone using @dnd-kit or react-beautiful-dnd, grid layout

Template Cards: Interactive hover lift (hover:shadow-lg) with orange border highlight on select

ExportButtons: Styled as cards, with icons, hover:bg-orange-500

🎯 Goals Page
SetGoalForm: Clean, compact UI with field grouping

GoalTracker: Timeline progress layout (flex justify-between), milestone dots

GoalStatusTag: Badge variants (Draft, In Progress, Achieved)

👤 Profile & Settings
Forms: Accordion panels for sections like business info, billing, notifications

PreferenceToggles: Switches for theme (light/dark), currency, notifications

IntegrationSettings: List of connected services, styled with pill buttons and icon tags

🧠 Context & State Management
Use React Context or Zustand to handle global state (e.g., dark mode toggle, user data).

Persist preferences like theme in localStorage.

🌟 Optional UI Polish
react-toastify for alerts

react-loading-skeleton for data fetching states

Tailwind Plugins: @tailwindcss/forms, @tailwindcss/aspect-ratio

Theme Toggle Button: Top-right switch (🌙/☀️), toggles class="dark"
