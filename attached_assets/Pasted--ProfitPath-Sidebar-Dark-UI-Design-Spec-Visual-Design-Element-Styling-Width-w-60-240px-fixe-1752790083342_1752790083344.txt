🧩 SidebarLink.jsx – Reusable Navigation Item
jsx
Copiar código
import { useLocation, Link } from 'react-router-dom';

const SidebarLink = ({ icon, label, path }) => {
  const location = useLocation();
  const isActive = location.pathname === path;

  return (
    <Link
      to={path}
      className={`flex items-center gap-2 px-3 py-2 rounded-md transition-all
        ${isActive ? 'bg-[#fd7014] text-white font-medium' : 'hover:bg-[#37414f] text-gray-300'}`}
    >
      <span>{icon}</span>
      <span>{label}</span>
    </Link>
  );
};

export default SidebarLink;
💡 Notes on Styling Decisions
🔶 Accent Usage:
Only the active link and logo use #fd7014.

Hover states use #37414f to stay subtle and clean.

🧭 Visual Hierarchy:
Section headers (text-xs, uppercase, text-gray-400) clearly define groups.

Icons on left, text on right—aligned for readability and scannability.

🧑‍💼 Footer:
Logout is deliberately separated by border-t for visibility without distracting.

✅ Final Result
Dark, minimal, elegant sidebar that guides user flow without overwhelming.

Matches ProfitPath’s identity: analytical, modern, business-focused.

Fully responsive-ready if needed (use hidden md:block to collapse on mobile).

Easily extendable for notifications, quick links, or theme toggles.
